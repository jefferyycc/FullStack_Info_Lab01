1. Change Directory: cd abc
2. Create a Directory: mkdir abc
3. Rename a Directory: mv abc xyz
4. List the contents of a directory: ls
5. Move a file to a new location: mv test.txt ..
6. Copy a file to a new location: cp text.txt ..
7. Search for a string in a file: grep "foo" test.txt
8. Access the shell of a remote machine: ssh remote_host
9. Remove a directory folder: rmdir abc
